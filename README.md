# Healync - Doctor Appointment Manager

A modern doctor appointment booking platform that bridges the gap between
top-tier medical specialists and patients' immediate health needs.

🌐 **Live Site:** [https://healync.vercel.app](https://healync.vercel.app)

---

## Features

- **Instant Appointment Booking** - Browse top-rated specialists and book online
  or in-person appointments in real time, skipping the waiting room entirely.
- **Top Rated Specialists** - Discover certified doctors across multiple
  specialties (Cardiology, Neurology, Psychiatry, and more) with experience
  details, hospital affiliation, and consultation fees.
- **Medical Companion Service** - A trained professional accompanies patients
  during hospital check-ins, medical tests, and prescription consultations for
  complete peace of mind.
- **Personal Dashboard** - Manage all your appointments and health records in
  one place through a dedicated user dashboard.
- **Daily Health Insights** - Receive practical wellness tips and healthy habit
  guides to help users stay proactive about their health every day.

## Getting Started

```bash
git clone https://github.com/tawchifulislam/healnyc.git
cd healnyc
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app locally.
